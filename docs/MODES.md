## Что показывает статистика

- При работе только Оптимизатора отображается трафик компьютера.
- При включённом SecureLink отображается трафик VPN-туннеля.
- TurboDNS влияет на статус защищённого DNS, но не подменяет общий счётчик сетевого трафика.
- Время сессии отсчитывается для активной защиты.
- Доступность сервисов проверяется независимо от наличия VPN.

## Рекомендуемый порядок диагностики

1. Проверить работу без сторонних VPN и сетевых фильтров.
2. Включить только Оптимизатор и Game Filter.
3. Проверить активную стратегию.
4. Выполнить встроенную диагностику.
5. Запустить автоподбор.
6. Только после этого отдельно проверить SecureLink или TurboDNS.

Не меняйте несколько модулей одновременно во время диагностики: иначе сложно определить источник проблемы.
