# StabiLink Desktop v3.1.0

17 июля 2026

Обновление сосредоточено на трёх вещах: стабильности SecureLink, скорости работы Оптимизатора и поведении приложения, когда серверы StabiLink временно недоступны.

## Главное

- Переключение SecureLink больше не перезапускает Оптимизатор — активные звонки и голосовые сессии Discord не обрываются.
- Оптимизатор запускается быстрее: убраны устаревшие сетевые проверки и блокирующие обращения к GitHub.
- Phantom гарантированно используется как основная встроенная стратегия, в том числе при первом запуске.
- Появился автономный режим: если серверы StabiLink недоступны, приложение всё равно запускается, а Оптимизатор и локальные функции остаются доступны.
- Обновления списков проверяются по SHA-256 и устанавливаются атомарно.

## SecureLink

- Интерфейс отображает реальное состояние туннеля.
- Если туннель аварийно завершился, статус автоматически меняется на «Отключён».
- Готовность сетевого адаптера VPN определяется быстрее.
- Ожидание подключения прекращается сразу, если процесс SecureLink завершился с ошибкой.
- Улучшено удаление зависших и скрытых сетевых адаптеров.
- Добавлена резервная настройка DNS для проблемных конфигураций Windows 10.
- Исправлены повторяющиеся DNS-адреса.
- Улучшено восстановление сети после неудачного подключения.

## Оптимизатор и Phantom

- Оптимизатор постоянно привязан к физическому интернет-интерфейсу.
- Переключение между Оптимизатором и SecureLink больше не требует перезапуска обработки трафика.
- Активные Discord UDP/RTC-соединения сохраняются при включении SecureLink.
- Обновление доменных списков и IP-диапазонов переведено на инфраструктуру StabiLink.
- Загруженные файлы проверяются по SHA-256 и размеру.
- Обновления применяются атомарно: повреждённый или недокачанный файл не заменит рабочий.
- При недоступности серверов StabiLink используется резервная загрузка через GitHub.
- Если недоступны оба источника, применяются последние корректные локальные списки.

## Автономный режим

Если серверы StabiLink временно недоступны, приложение продолжает запускаться и не разлогинивает пользователя. Оптимизатор, Phantom, Game Filter и TurboDNS работают как обычно.

SecureLink в это время не запускается: VPN требует актуальной проверки подписки, и StabiLink не включает его по устаревшим данным. После восстановления связи с сервером режим снимается автоматически.

## Подписка и авторизация

- Удалён устаревший PRO-кэш с диска.
- SecureLink нельзя запустить по просроченным данным о подписке.
- Истечение PRO корректно переводит пользователя на FREE после проверки сервером.
- Временная недоступность API больше не приводит к выходу из аккаунта или удалению настроек.
- Все способы запуска SecureLink — главное окно, расширенный режим и трей — используют единую проверку доступа.

## Интерфейс

- Компактное окно адаптировано для экранов 1366×768 и повышенного масштабирования Windows.
- На небольших экранах добавлена вертикальная прокрутка.
- Основная кнопка управления больше не уходит за границы экрана.
- Статус SecureLink синхронизирован между главным окном, компактным окном и треем.
- Исправлено обновление интерфейса после изменения авторизации и подписки.
- Версия приложения отображается из единого источника.

## Производительность и стабильность

- Удалены устаревшие компоненты телеметрии и неиспользуемых стратегий.
- Снижено количество фоновых операций и повторных обновлений статистики.
- Исправлена повторная загрузка метрик и истории сессий.
- Данные старше 90 дней очищаются автоматически.
- Статистика сохраняется надёжнее при одновременной записи.
- Исправлено завершение приложения после выхода из аккаунта: выполняется корректная остановка SecureLink, Оптимизатора и сетевых компонентов.
- Улучшено логирование ошибок подключения и восстановления сети.

## Установка

1. Скачайте `StabiLink-Desktop-v3.1.0-win-x64.zip`.
2. Сверьте SHA-256.
3. Полностью распакуйте архив.
4. Запустите `StabiLink.Desktop.exe` и подтвердите права администратора.

Не запускайте приложение непосредственно из ZIP.

Обновление поверх 3.0.0 сохраняет настройки и авторизацию.
