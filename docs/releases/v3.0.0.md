# StabiLink Desktop v3.0.0

15 июля 2026

Это самое крупное обновление Desktop-приложения StabiLink за последнее время. Мы полностью переработали интерфейс, навели порядок внутри приложения и сделали ежедневное управление заметно понятнее.

## Главное

- Полностью новое компактное окно.
- Новый StabiLink Control Center.
- Обновлённые профиль, авторизация, настройки, трей и уведомления.
- Более плавный график трафика и корректная статистика разных режимов.
- Улучшенные Phantom и Game Filter.
- SecureLink VPN можно включать вместе с оптимизацией одной кнопкой.
- Автоподбор стратегии теперь можно отменить.
- Необязательное обновление можно отложить.
- Безопаснее QR-вход и диагностические журналы.

## Новый интерфейс

CompactWindow показывает только текущее состояние, активную стратегию, время сессии и статусы VPN/DNS/DPI. Для подробной работы доступен Control Center с девятью разделами: обзор, подключение, стратегии, статистика, диагностика, журналы, утилиты, настройки и сведения о приложении.

## Соединение и статистика

- Phantom получил обновлённые настройки обработки соединения.
- Game Filter включён по умолчанию для совместимых игровых и голосовых сценариев.
- Трафик корректно отображается при работе только Оптимизатора, с SecureLink или с TurboDNS.
- График реального времени стал плавнее.
- Исправлена статистика доступности сервисов без обязательного запуска VPN.

## Удобство

- Настройки, трей и автозапуски сохраняются после перезапуска.
- Добавлена настройка совместного запуска Оптимизатора и SecureLink для PRO.
- Автоматический подбор стратегии можно отменить без зависшей фоновой операции.
- Все popup-уведомления приведены к единому стилю.
- Убраны двойные и ошибочные уведомления.
- Исправлены прокрутка, размеры карточек и обрезание текста.

## Обновления

Обычное обновление больше не закрывает приложение принудительно. Сначала появляется закреплённое уведомление и кнопка установки. Обязательное обновление применяется только для версии ниже минимально поддерживаемой.

Окно обновления показывает подготовку, загрузку, распаковку, установку и перезапуск.

## Надёжность и безопасность

- Улучшено завершение сетевых компонентов при выключении модулей, выходе из приложения и завершении Windows.
- Снижено количество зависших процессов.
- Чувствительные данные автоматически маскируются в журналах и диагностических архивах.
- Исправлены несколько гонок фоновых операций и обработки ошибок.

## Установка

1. Скачайте `StabiLink-Desktop-v3.0.0-win-x64.zip`.
2. Сверьте SHA-256.
3. Полностью распакуйте архив.
4. Запустите `StabiLink.Desktop.exe` и подтвердите права администратора.

Не запускайте приложение непосредственно из ZIP.
