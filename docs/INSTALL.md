# Установка StabiLink Desktop

1. Скачайте архив актуальной версии из Releases или личного кабинета StabiLink.
2. Сверьте SHA-256 архива со значением в описании релиза.
3. Распакуйте архив в отдельную папку, доступную текущему пользователю.
4. Запустите StabiLink и подтвердите запрос прав администратора.
5. Авторизуйтесь и пройдите короткую первичную настройку.

Не запускайте приложение непосредственно из ZIP-архива и не заменяйте отдельные DLL/EXE файлами из сторонних источников.

## Обновление

StabiLink сообщит о новой версии внутри приложения. Необязательное обновление можно отложить; критическое обновление потребуется установить для продолжения работы.

Пользовательские настройки и данные авторизации должны сохраняться при штатном обновлении. Перед ручной заменой каталога закройте приложение через трей.
